# Sentiment Analysis (RU/DE)

## Описание проекта

Данный проект представляет собой прототип системы анализа тональности текстов на русском и немецком языках.

Система классифицирует текст по трём категориям:
- `positive` — положительная тональность;
- `negative` — отрицательная тональность;
- `neutral` — нейтральная тональность.

В основе решения используется мультиязычная трансформерная модель **XLM-RoBERTa**, дообученная на размеченном наборе данных.

## Основные возможности

Проект реализует следующий функционал:
- загрузку и предобработку текстовых данных;
- дообучение модели анализа тональности;
- предсказание тональности в консольном режиме;
- запуск веб-интерфейса на основе Gradio;
- автоматическую проверку проекта с помощью GitHub Actions;
- вывод архитектуры модели с помощью `torchinfo`;
- визуализацию архитектуры модели с помощью `Netron`.
