# Sentiment Analysis (RU/DE)

## Описание проекта

Данный проект представляет собой прототип системы анализа тональности текстов на русском и немецком языках.

Система классифицирует текст по трём категориям:

* positive — положительная тональность
* negative — отрицательная тональность
* neutral — нейтральная тональность

В основе решения используется мультиязычная трансформерная модель XLM-RoBERTa.

---

## Используемые технологии

* Python
* PyTorch
* Hugging Face Transformers
* Gradio
* scikit-learn
* pandas

## Установка

1. Клонировать репозиторий:

git clone https://github.com/dishubnyakova/sentiment-analysis-ru-de.git
cd sentiment-analysis-ru-de

2. Установить зависимости:

pip install -r requirements.txt

---

## Обучение модели

Перед использованием необходимо обучить модель:

python train.py

После обучения модель сохраняется в папке model/.

## Результаты

В ходе тестирования были получены следующие метрики:

* Accuracy ≈ 0.33
* F1-score ≈ 0.23

Модель демонстрирует базовую работоспособность, однако требует дальнейшего улучшения за счёт увеличения объёма данных.
